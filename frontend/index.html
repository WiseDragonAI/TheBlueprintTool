Telemetry

Controller path

    Thread

    Notes

    No thread selected

    Delete selected zone?

    Keys

    A
    Open or focus the thread panel.
    X
    Start or stop the active voice note.
    Esc
    Cancel voice capture, close thread tooling, or clear selection.
    Del
    Confirm deletion for the selected card, zone, or group.
    Ctrl+C
    Copy the selected cards, zones, and groups.
    Ctrl+V
    Paste the copied selection.
    Ctrl+D
    Resize selected cards to their content.

    Runbook

    Workspace Server

    Start CoreV2 from the target workspace cwd so the active .blueprinttool directory resolves from that workspace.

    env PORT=4174 /home/jbb/dev/EditorBP/CoreV2/bin/blueprinttool-server.mjs

    Card Images

    Markdown image URLs under .blueprinttool render in cards. Adjacent standalone images become a carousel, and width resizing keeps the loaded image aspect ratio while persisting card.imageSizes[src].

    ![Alt](.blueprinttool/ui-mockups/example.png)

    Voice Notes

    Voice config stays workspace-local in .blueprinttool/.settings.json. Uploaded audio remains under .blueprinttool/voice-uploads/.
