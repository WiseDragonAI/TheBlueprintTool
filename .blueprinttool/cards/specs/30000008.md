Canvas rendering must keep pointer interaction responsive at wide zoom-out levels by switching the canvas into low-detail mode below 0.35 scale and overview-detail mode below 0.18 scale. Low-detail mode removes expensive shadows, description blocks, buttons, hashes, and relationship labels while preserving geometry, selection affordances, readable card titles, and card label chips. Overview-detail mode keeps card titles readable with the viewport counter-scale rule instead of rendering unreadable description columns.
