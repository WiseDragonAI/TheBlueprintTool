Ledger cards can declare visual tags using a `labels` or `tags` string array. Non-empty labels render as compact chips in the card's top-right corner at normal detail. The chip background uses the card's own `--card-zone-color`, so labels stay visually tied to the card color/zone context. Labels are decorative metadata on the card, not relationships, identities, or runtime state. In low-detail zoom, labels remain visible with viewport counter-scaling, centered horizontally and vertically inside the card, while internal hash ids stay hidden.
